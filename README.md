# CrowdFundHub - Crowdfunding Platform

A full-stack crowdfunding platform built with Next.js, Express.js, MongoDB, and Node.js (MERAN Stack).

## Features

- Responsive design optimized for mobile, tablet, and desktop views
- Three user roles: Supporter, Creator, and Admin with tailored dashboards
- Secure authentication with email/password and Google Sign-In
- Role-based private routes with persistent sessions after page reload
- Interactive homepage with Swiper slider, top-funded campaigns, and testimonials
- Campaign management system with create, update, delete, and refund functionality
- Real-time contribution tracking with approve/reject workflow
- Credit purchase system with Stripe payment integration
- Creator withdrawal system (20 credits = $1 conversion rate)
- Admin dashboard for user management, campaign moderation, and withdrawal processing
- Notification system with floating pop-up for all user actions
- Pagination on My Contributions page for better data navigation
- Advanced search and filter functionality for campaigns by category
- Reporting system for suspicious or fraudulent campaigns
- Animation and smooth transitions throughout the platform
- Environment variables for secure configuration management
